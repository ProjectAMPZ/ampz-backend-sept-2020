## Requirements and Installation

To install and run this project you would need to have installed:
- Node Js
- Git

To run:
```
$ git clone https://github.com/ProjectAMPZ/ampz-backend-sept-2020.git
$ cd ampz-backend-sept-2020
$ npm install
$ npm run dev
```

## Testing
```
$ npm test
```
